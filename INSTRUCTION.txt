Приложение создал Unforg1ven (Varguss на github.com).

Приложение являет собой пользовательский веб-интерфейс для получения информации об банах и джоббанах, выданных на сервере Tau Ceti Gamma (однако, легко конфигурируется и под другие сервера, где база данных имеет одинаковую сигнатуру.

Для того, чтобы изменить настройки подключения к БД, достаточно заглянуть в файл database.properties.

Пример содержимого файла:
### TEST SETTINGS ###
url=jdbc:mysql://localhost:3306/tgstation
user=root
password=1
ban_table=erro_ban

Проект является WEB-приложением и должно устанавливать на Tomcat 9 версии, на других версиях тесты не проводились.

Рекомендуется задать стандартный путь к приложению в Tomcat ("/").

Небольшая фича: так как DAO приложения хранит в себе кэш, он должен обновляться. В коде есть специальная демон-нить, которая каждый час автоматически обновляет кэш. Однако, если нужно обновить кэш вне очереди, то
можно воспользоваться следующей ссылкой:

хост:порт/cash?password=diesiegePassword322

Тогда кэш будет обновлен сразу.

Важно: не использовать русские директории на пути к приложению.

Есть два способа запуска приложения: 1 - командой через консоль, 2 - ручной деплой на томкэт.

Способ №1 - командой через консоль:

Откройте консоль в корне проекта, после чего введите команду: "mvn package tomcat7:run". Важно: для корректного выключения сервера нужно в консоли нажать CTRL+C для завершения процесса.
Если закрыть консоль, то сервер Tomcat будет висеть в процессах как java процесс.

Способ №2 - ручной деплой на томкэт:

Преимущество данного способа в том, что достаточно сделать все раз.

1) Открыть консоль в корневой папке проекта и выполнить команду "mvn package".
2) В папке target директорию UserBanInterface перенести в корневую_директорию_томкэта/webapps и переименовать в ROOT. Если ROOT уже был (и это не старая версия проекта), то запаковать и сохранить в корне томкэта или удалить.

После этого, можно будет запускать приложение через обычный запуск томкэта.
